Adds controls to directly add and remove tags from the Posts and Media lists within the admin area.

== Installation ==

1. Upload the `wp-quick-tags` folder to the `/wp-content/plugins/` directory
2. Activate 'Quick Tags' through the 'Plugins' menu in WordPress
3. Go to 'Posts > All Posts' or 'Media > Library' to start adding and removing tags directly from the lists
